# Инструкция по использованию эндпоинтов сервера

Сервер предоставляет четыре эндпоинта для взаимодействия с шифром Roller Cipher. Все запросы выполняются по протоколу HTTPS. Базовый адрес сервера:

`https://roller.worldz.tech`

## 1. GET /flag

**Описание:** Получение зашифрованного флага — основной целевой шифротекст задания.
Это именно то сообщение, которое вам нужно расшифровать после нахождения ключей n и m.

**Параметры:** отсутствуют

**Ответ:** JSON-массив целых чисел (список байтов после шифрования)

**Пример ответа:**
```bash
[45, 178, 93, 12, 241, 67, 89, ...]
```

**Python**

```python
import requests

response = requests.get("https://roller.worldz.tech/flag")

encrypted_flag = response.json()

print(encrypted_flag)
```

## 2. GET /sample
**Описание:** Получение демонстрационного примера. Возвращает открытый текст (sample_flag) и его зашифрованную версию с использованием тех же ключей n и m.
Полезно для проверки понимания шифра и тестирования расшифровки.

**Параметры:** отсутствуют

**Ответ:** строка в формате "открытый_текст: [список_чисел]"

**Пример ответа:**
{sample_flag}: [97, 99, 111, 105, 104, ...]

**Python**
```python
import requests

response = requests.get("https://roller.worldz.tech/sample")

print(response.text)
```

## 3. POST /encrypt
**Описание:** Шифровальный оракул (encryption oracle). Позволяет зашифровать любой текст, используя те же секретные ключи n и m.
Это основной инструмент для анализа и восстановления ключей.

**Параметры (в теле запроса, JSON):**

* plain_text — строка, которую нужно зашифровать (обязательно)
* n — целое число (сдвиг первой последовательности)
* m — целое число (сдвиг второй последовательности)

**Ответ:&& список целых чисел (результат шифрования)

**Пример тела запроса:**
```json
{
  "plain_text": "test message",
  "n": 7,
  "m": 3
}
```

**Пример запроса в Python:**
```python
import requests

data = {
    "plain_text": "aaaaaaaaaaaaaaaaaaaa",
    "n": 10,
    "m": 20
}
response = requests.post("https://roller.worldz.tech/encrypt", json=data)
print(response.json())  # [23, 45, 178, ...]
```

## 4. POST /decrypt
**Описание:** Расшифровочный оракул. Позволяет расшифровать любой список байтов, используя указанные n и m.
Используйте его для проверки своих гипотез о ключах.

**Параметры (в теле запроса, JSON):**

* bytes — список целых чисел (шифротекст, который нужно расшифровать)
* n — целое число
* m — целое число

**Ответ:** строка (расшифрованный текст)

**Пример тела запроса:**
```json
{
  "bytes": [97, 99, 111, 105, 104],
  "n": 2,
  "m": 1
}
```

**Пример запроса:**

```python
import requests

data = {
    "bytes": encrypted_flag,  
    "n": 123,                 
    "m": 456
}
response = requests.post("https://roller.worldz.tech/decrypt", json=data)
print(response.json())  # {flag} если ключи верные
```
